# Project Summary

This project demonstrates the practical implementation of **Object Oriented Programming (OOPS)** concepts in Python through real-world problem simulations.

Each module models an industry-based system using classes, attributes, and methods to reflect structured programming practices used in professional software development.

# Core OOPS Concepts Applied

| Concept | Implementation |
|----------|---------------|
| Class Creation | Defined separate classes for each system |
| Constructor | Used `__init__()` to initialize object attributes |
| Instance Variables | Stored object-specific data |
| Methods | Created behavior-based functions inside classes |
| Object Creation | Instantiated objects to simulate real-world usage |

# System Modules Implemented

## 1️ Smart Light Controller
- Controls ON/OFF state of a smart light  
- Displays current status  
- Industry Use: Smart Home Automation  

## 2️ Employee ID Card System
- Stores employee name, ID, and department  
- Displays formatted ID card  
- Industry Use: Corporate HR Systems  

## 3️ Mobile Contact Record
- Stores contact name and phone number  
- Displays saved contact information  
- Industry Use: Mobile Applications  

## 4️ Product Price Tag Generator
- Stores product details  
- Prints formatted price tag with currency  
- Industry Use: Retail & Billing Systems  

## 5️ Movie Rating Display System
- Stores movie name and rating  
- Displays rating in standard format  
- Industry Use: Streaming Platforms  

## 6️ Delivery Address Manager
- Stores customer and delivery address  
- Prints delivery details  
- Industry Use: Logistics & Courier Platforms

This project showcases structured OOPS implementation in Python by translating real-world systems into modular, reusable class-based programs.
